First read the execution rules in [execute_plan.instructions.md](../instructions/execute_plan.instructions.md) -- these are INVIOLABLE.

Open the specified planfile and work through it **task by task**:

1. Identify next open `- [ ]` task in the current category
2. Implement task completely -- highest detail, production quality
3. Set checkmark IMMEDIATELY in planfile: `- [ ]` to `- [x]`
4. Next task -- back to step 1

**Never** work on multiple tasks simultaneously. **Never** set checkmarks in advance.
